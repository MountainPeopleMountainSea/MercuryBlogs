重学vue2.0-内部指令

Nov 2, 2019 | vue

vue之前已经学过了，但是最近一段时间在使用react，有一段时间没有使用过vue了。对于一些基础知识也有些模糊，打算从基础开始，再学一遍vue

v-if、v-else和v-show

v-if用来判断是否加载html的某个dom，v-if为真时显示该内容，否则显示v-else的内容。

v-show用来调整css的display属性，此时的dom已经加载。v-show=true，会给该dom添加一个css属性：display:none

v-if和v-show的区别：

v-if:判断是否加载dom，减轻服务器压力

v-show:此时已加载dom，只是被隐藏

v-for：解决模板循环加载问题

v-text和v-html

通过在html中显示data的值，在某些情况下，这样做也是有弊端的，比如当网速很慢或js出错时，就会暴露这个值。

v-text就很好的解决了这个问题，

这样即使出错，用v-text也显示不出来。若在js中加了html标签，此时就需要v-html
