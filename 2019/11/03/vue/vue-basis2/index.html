1.1 自定义指令中传递的三个参数

  • el:指定所绑定的元素，可以直接操作dom

  • binding：对象，包含了指令的信息（包括指令的名称，指令的值等等）

  • vnode：Vue编译生成的虚拟节点

需注意：在声明一个自定义的指令时，指令中不可包含大写字母
