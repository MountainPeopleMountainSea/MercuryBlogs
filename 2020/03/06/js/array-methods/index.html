array_methods.md

Mar 6, 2020 | JavaScript

在JavaScript里，数组也是一个对象，js也给数组添加了很多实用的方法。方法有很多，所以我这里记录一下常用的方法，方便查询

第一部分主要是对数组的增删、截取

  • 1.unshift(element)：向数组头部添加一个元素
  • 2.shift()：删除数组的第一个元素
  • 3.push(element)：向数位尾部添加元素
  • 4.pop()：删除数组最后一个元素
  • 5.splice(x,y)：删除从数组索引x开始的y个元素
  • 6.slice(x,y)：截取方法，会返回一个数组索引x开始的y个元素

下面这些主要是数组的迭代、遍历方法

现在定义一个判断元素是否为偶数的箭头函数，下面的方法都用这个作为栗子
const isEven = x => x%2 === 0

  • 7.every(isEven)：遍历所有元素是否为偶数，有一个不是就返回false，全对才返回true
  • 8.some(isEven)：与every相反，有一个满足条件就返回true，全错才返回false
  • 9.forEach( x => console.log(x%2 === 0))，与执行for循环相同，没有返回值
  • 10.map(isEven)：校验每一个元素，返回对应的校验结果true或false的数组
  • 11.filter(isEven)：与map相似，不同的是，它返回的是满足条件的元素组成的数组
  • 12.for of循环方法：栗子：for(const n of arr){console.log(n%2 === 0)}
  • 13.reduce((pre,next) => pre+next)：数组求和快捷方法

下面主要记录数组的查询方法

  • 14.indexOf(element)：返回与参数匹配的第一个元素的索引
  • 15.lastIndexOf(element)：返回最后一个像匹配的索引
  • 16.find(cb)：传入一个回调函数，返回第一个满足回调函数条件的元素，es6加入
  • 17.findIndex(cb)：传入一个回调函数，返回第一个满足的索引，es6加入
  • 18.includes(element)：判断数组中是否存在该元素，有就为true，没有就是false，es7加入

数组排序

  • 19.sort(cb)：可传入回调函数或者不传，不传默认按ASCII码表来进行排序，常用的数值排序sort((a,b) => a-b)

其它常用的方法

  • 20.concat：数组合并，用法：arr1.concat(arr2)，返回一个数组
  • 21.join(s)：将数组连成一个字符串，参数可传任意的字符来连接
  • 22.split(‘’)：字符串转数组，注意转化后都是string类型的元素
